## Command-Line Tools

- [Homebrew](Homebrew.html) - useful for installing things on Steam Deck and MacOS (and Typst on Raspberry Pi)
- [topgrade](topgrade.html) - upgrade everything
- [Zellij](https://zellij.dev/) - better `tmux` (better `screen`)
- [tldr](https://tldr.sh/) - too long; didn't read `man` pages with examples
- [zsh](https://wiki.archlinux.org/title/zsh) - better `bash` (better‚Ä¶ `dash`?) with [omz](https://ohmyz.sh/) and [powerlevel10k](https://github.com/romkatv/powerlevel10k)
- [bat](https://github.com/sharkdp/bat) - better `cat`
- [eza](https://github.com/eza-community/eza) - better `ls`
- [yazi](https://yazi-rs.github.io/) - terminal file manager
- [spotdl](https://github.com/spotDL/spotify-downloader) - download songs (and playlists) using Spotify metadata from YouTube Music
- [btop](https://github.com/aristocratos/btop) - better `htop` (better `top`)
- [Atuin](https://atuin.sh/) - better shell history
- [thefuck](https://github.com/nvbn/thefuck) - correct errors in previous console commands

## Desktop (Linux üêß)

- Always use a password manager
- [KDE](https://kubuntu.org/) (via [Kubuntu](https://kubuntu.org/))

## Other

Programs/apps that are cool but don't use often but are worth mentioning:

- [OpenFoodFacts](https://f-droid.org/en/packages/openfoodfacts.github.scrachx.openfood/) App - Look up food ingredients, allergens, nutrition facts
	- Relies on [Open Food Facts](https://world.openfoodfacts.org/discover), a public and open database for food products. Open databases are **great**.
	- If what you scanned isn't available, it provides a very easy and quick way to add it to the database thereby improving the data for everyone. Also good for updating out of date facts.
	- I live in Ireland and it worked surprisingly well for products here. Anything that wasn't there I could add easy and quickly.
- [MakeACopy](https://f-droid.org/en/packages/de.schliweb.makeacopy/) - privacy focused document/text scanner
